# [Advent of Code - Day 6](https://adventofcode.com/2019/day/6)
Build [AdventOfCodeDay6](https://github.com/BigETI/AdventOfCode/tree/master/AdventOfCodeDay6), then get the input from [https://adventofcode.com/2019/day/6/input](https://adventofcode.com/2019/day/6/input) and save as `orbital-map.txt`.

After that execute your build to get results.

## Part 1
```BASH
AdventOfCodeDay6 < orbital-map.txt
```

## Part 2
```BASH
AdventOfCodeDay6 YOU)SAN < orbital-map.txt
```
